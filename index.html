Registack AIR

Agent Detector CLI

Local-only detector for AI-agent candidates, runtime artifacts, Docker workloads, and Kubernetes manifests. Install from the terminal, select the default detection path by number, and emit AIR-compatible JSON.

macOS

Install and choose the default detection path interactively by number.

curl -fsSL https://registack.eu/cli/registack-agent-detector/install-macos.sh | bash

Linux

Install and choose the default detection path interactively by number.

curl -fsSL https://registack.eu/cli/registack-agent-detector/install-linux.sh | bash

Windows

Run the PowerShell installer and select the default detection path by number.

powershell -ExecutionPolicy Bypass -c "irm https://registack.eu/cli/registack-agent-detector/install-windows.ps1 | iex"

Verify

Once installed, verify the CLI and run the saved default detection path explicitly.

registack-agent-detector --version
registack-agent-detector --scan-default --output json

Operator Notes

No manual path typing is required during installation. The installer presents a numbered list of valid detection paths and stores the selected default path for later scans.
